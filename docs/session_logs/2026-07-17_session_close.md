## Зафиксированные решения

1. Логотип Элдокс является ядром визуальной палитры.
2. Все визуальные решения должны подстраиваться под логотип: палитра, свет, формы, неоновая технологичность, геометрия контуров.
3. Для продающего визуального оформления по умолчанию используются `PNG/JPG/WEBP`.
4. `SVG` для художественных/маркетинговых визуалов запрещен без отдельного разрешения владельца.
5. В продуктовых ассетах Saby людей по умолчанию не используем.
6. Если люди нужны по смыслу, они должны быть локально правдоподобны для русскоязычной аудитории. Демонстративная западная diversity-эстетика не подходит.
